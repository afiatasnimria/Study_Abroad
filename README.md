# 🌍 Study Abroad Inquiry System

A web-based platform designed to help students explore international study opportunities, connect with education consultants, and manage their study abroad applications efficiently.

This project simplifies the **study abroad journey** by integrating university search, consultation booking, document management, and application tracking into a single system.

---

# 📌 Project Overview

Studying abroad involves multiple complex steps such as selecting universities, preparing documents, applying for admissions, and managing visa processes. Many students struggle to gather reliable information and track their application progress.

The **Study Abroad Inquiry System** provides a centralized platform where students can:

- Search universities and programs
- Submit inquiries
- Upload academic documents
- Book consultations with advisors
- Track application progress

The system also provides an **admin dashboard** for managing student inquiries and applications.

---

# 🎯 Objectives

The main objectives of this project are:

- Simplify the process of applying to international universities
- Provide centralized information about study destinations
- Improve communication between students and consultants
- Enable efficient management of student inquiries
- Provide real-time application tracking

# 🏗 System Architecture

The system follows a **client-server architecture**:

```

      │
      ▼
Web Interface (Frontend)
      │
      ▼
Application Server (Backend)
      │

```
